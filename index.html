September 12, 2012
V2.6
Updated a lot of the code to comply with Chrome's stricter security standards. Interface has been revamped; it should be more intuitive now. If something isn't working right, please let me know, and I will try to fix it as soon as possible.

Keyboard shortcuts are now natively supported. You can set them up under the hotkeys section. If you try to enable them, you should be prompted for additional permissions. The old ChromeToPaper Keyboard Shortcuts extension is now unneeded.
March 20, 2011
V2.5.4
You can now choose an icon via the appearance section! Whatever icon you choose should immediately appear in the action bar (let me know if it does not). There's currently no way to choose an icon from your own computer (due to permissions), but I am completely open to user-submitted icons.
